# Funnel Clone Deployment

Este projeto foi configurado para ser implantado na **Vercel**.

## Como fazer o deploy:

1.  Tenha a [Vercel CLI](https://vercel.com/cli) instalada.
2.  No terminal, abra a pasta do projeto: `cd "c:\Downloaded Web Sites\larpzwallet.com\inleaddigiyli9.arktrix.app"`
3.  Execute o comando: `vercel deploy --prod`

Ou conecte seu repositório Git (GitHub, GitLab, Bitbucket) à Vercel e ela detectará as configurações automaticamente.

## Detalhes técnicos:
- **vercel.json**: Faz o redirecionamento automático da página principal (`/`) para a página do funil em `/posiciones-enloquecedoras/index.html`.
- **package.json**: Ajuda a Vercel a reconhecer o tipo de projeto.
